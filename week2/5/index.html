Game TEKNIKSERANG dengan bonus ramalan CINTA CLI (Command Line Interface)

Ini adalah aplikasi game sederhana yang menarik dan mudah jika dibandingkan dengan lainnya.. game ini berjakan di via (GUI) Command Line Interface.

Untuk menggunakanya, lihat Console CHROME di INSPECT.

Fitur aplikasi

Aplikasi ini memiliki fungsi-fungsi sederhana, yakni :

  • Sesi yang digunakan
  • Perhitungan game
  • Konversi Nilai Abjad
  • Rumusan Ramalan
  • Penjelasan Ramalan
  • Memulai Ramalan
  • Menghapus Ramalan

Pseudocode :

Fungsi intro() atau penjelasan permainan:

  • DECLARE variable ‘tokoh’ dengan memilih antara penembak atau samurai
  • IF variable ‘tokoh’ sama dengan penembak THEN
    • DECLARED variable ‘kekuatan’ berisikan shortgun dan gunkong
  • ELSE IF variable tokoh sama dengan samurai THEN
    • DECLARED variable kekuatan berisikan blidecar dan blidekong
  • ELSE
    • CALL fungsi clear()
  • DISPLAY mengucapkan selamat datang kepada variable tokoh dan menjelaskan
  • kekuatannya berdasarkan variable kekuatan dan aturan pakai kekuatan untuk variable tokoh.
  • DISPLAY menjelaskan untuk melanjutkan permainan jika mengetikkan sesi(1) di dalam console
Fungsi sesi(indexsesi) atau tahapan permainan:
  • IF variable indexsesi sama dengan 1 THEN
    • DECLARED variabel soal dengan 45
    • DISPLAY soal dari permainan ini dan DECLARE variable soal
    • CALL fungsi perang dengan menyertakan variabel soal dan score dan DECLARE variabel hasil
    • IF variable hasil sama dengan 0 THEN
      • DISPLAY mengucapkan selamat karena sudah memenangkan permainan
      • DISPLAY bonus pemenang untuk mengramal pasangan dengan mengetikkan ramal() di dalam console
    • ELSE
      • DISPLAY Semangat melanjutkan lagi permainan
Fungsi perang(jawabanSoal, nilaiKekuatan monster) atau tahapan menghitung nilai tokoh melawan monster:
  • DECLARE variable score sama dengan variable nilaiKekuatan
  • SPLIT variable jawabanSoal dengan pemisah ‘’ dan DECLARE variable c
  • IF panjang dari variable c kurang dari sama dengan 8 THEN
    • FOR setiap variable c
      • IF variable c dalam array pengulangan sama dengan 2 THEN
        • IF variable c dalam array pengulangan dikurangin 1 dan 2 sama dengan 1 THEN
          • DECLARED variable score sama dengan variable score dikurang 10
          • DISPLAY saudara variable tokoh sudah menyerang monster dengan kekuatan kedua
        • ELSE IF variable c dalam array pengulangan dikurangi 1 dan 2 tidak sama dengan 1 THEN
        • DISPLAY maaf kekuatan senjata belum aktif
    • ELSE IF variable c dalam array pengulangan sama dengan 1 THEN
      • DECLARED variable score sama dengan variable score dikurang 5
      • DISPLAY saudara variabale tokoh menyerang monster dengan kekuatan pertama
    • RETURN CALL variable score
  • ELSE
    • DISPLAY maaf inputan anda lebih dari 8
    • RETURN CALL variable score
Fungsi Ramal() atau meramal pasangan:
  • CLEAR console
  • DISPLAY nama lengkap anda dan DECLARED variable namaCowok
  • DISPLAY nama lengkap pasangan anda dan DECLARED variable namaCewek
  • CALL fungsi hitungRamalan diikuti variabel namaCowok dan namaCewek
Fungsi hitungRamalan(namaCowok dan namaCewek) atau menghitung nilai dari nama:
  • SPLIT variable namaCowok dengan pemisah ‘ ‘ atau spasi dan DECLARED variabel splitCO
  • SPLIT variable namaCewek dengan pemisah ‘ ’ atau spasi dan DECLARED variable splitCE
  • FOR setiap variabel splitCO
    • SPLIT variable splitCO dengan array yang diulang dan DECLARED variable hurCO
    • CALL fungsi konvert disertai variable hurCO dengan array 0 dan DECLARED variable nilai
    • DECLARED variable nilaiCowok dengan penambahan variable nilaiCowok ditambah variable nilai
  • FOR setiap variabel splitCE
    • SPLIT variable splitCE dengan array yang diulang dan DECLARED variable hurCE
    • CALL fungsi konvert disertai variable hurCE dengan array 0 dan DECLARED variable nilai
    • DECLARED variable nilaiCewek dengan penambahan variable nilaiCewek ditambah variable nilai
  • DECLARED variable hasil dengan penambahan variable nilaiCowok ditambah nilaiCewek
  • CALL fungsi penjelasan disertai variable hasil
Fungsi penjelasan(nilaiRamalan) atau menyamakan nilai berdasarkan wijaya kusuma:
  • IF variable nilaiRamalan lebih kecil sama dengan 9 THEN
    • DISPLAY hasil ramalan wijaya kusuma
  • ELSE IF variable nilaiRamalan lebih kecil sama dengan 19dan variable nilaiAbjad lebih kecil sama dengan 10 THEN
    • DISPLAY hasil ramalan wijaya kusuma
  • ELSE IF variable nilaiRamalan lebih kecil sama dengan 29dan variable nilaiAbjad lebih kecil sama dengan 20 THEN
    • DISPLAY hasil ramalan wijaya kusuma
  • ELSE IF variable nilaiRamalan lebih kecil sama dengan 39 dan variable nilaiAbjad lebih kecil sama dengan 30 THEN
    • DISPLAY hasil ramalan wijaya kusuma
  • ELSE
    • DISPLAY hasil ramalan wijaya kusuma
Fungsi konvert(nilaiAbjad) atau mengkonversi huruf menjadi nilai menurut wijaya kusuma:
  • IF variable nilaiAbjad sama dengan a OR j OR s THEN
    • DECLARED variable nilai sama dengan 1
  • ELSE IF variable nilaiAbjad sama dengan b OR k OR t THEN
    • DECLARED variable nilai sama dengan 2
  • ELSE IF variable nilaiAbjad sama dengan c OR l OR u THEN
    • DECLARED variable nilai sama dengan 3
  • ELSE IF variable nilaiAbjad sama dengan d OR m OR v THEN
    • DECLARED variable nilai sama dengan 4
  • ELSE IF variable nilaiAbjad sama dengan e OR n OR w THEN
    • DECLARED variable nilai sama dengan 5
  • ELSE IF variable nilaiAbjad sama dengan f OR o OR x THEN
    • DECLARED variable nilai sama dengan 6
  • ELSE IF variable nilaiAbjad sama dengan g OR p OR y THEN
    • DECLARED variable nilai sama dengan 7
  • ELSE IF variable nilaiAbjad sama dengan h OR q OR z THEN
    • DECLARED variable nilai sama dengan 8
  • ELSE
    • DECLARED variable nilai sama dengan 9
  • RETURN variable nilai
